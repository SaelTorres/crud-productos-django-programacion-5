# CRUD de Productos con Django

Asignatura Sael Torres 30.980.467
Aplicación web monolítica desarrollada con Django 4.x que implementa un CRUD completo para la gestión de productos.

## Funcionalidades del CRUD 

- Crear productos
- Listar productos
- Ver detalle de un producto
- Editar productos
- Eliminar productos con confirmación

## Tecnologías utilizadas

- Python 3.x
- Django 4.x
- SQLite
- HTML
- Postman (para pruebas HTTP)

## Modelo de base de datos

Producto:
- nombre (CharField)
- descripcion (TextField)
- precio (DecimalField)
- stock (IntegerField)
